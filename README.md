# Rekols Dictionary

一款Deepin平台下的词典，基于Qt开发，使用有道词典API，占用内存小。

## 编译依赖

`sudo apt install g++ qt5-default libdtkcore-dev libdtkwidget-dev libqt5svg5-dev`

## 编译

* mkdir build
* cd build
* qmake ..
* make

## 运行

* ./redict
