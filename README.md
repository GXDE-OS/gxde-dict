# ReDict

Rekols词典，基于dtk开发的一款词典（有道API）

## 编译依赖

`sudo apt install g++ qt5-default libdtkcore-dev libdtkwidget-dev`

## 编译

* mkdir build
* cd build
* qmake ..
* make

## 运行

* ./redict
